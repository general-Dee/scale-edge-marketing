# Scale-Edge Marketing LTD

A modern e-commerce platform built for the Nigerian market, featuring Paystack payment integration, Meta Pixel tracking, and a seamless shopping experience.

## 🚀 Features

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Paystack** payment integration
- **Shopping Cart** with local storage
- **Meta Pixel** tracking
- **Mobile responsive** design
- **Dark mode** support

## 📦 Products Categories

- ☀️ Solar Essentials
- 🧴 Skincare
- 🏠 Home Solutions
- 🌸 Fragrance
